# Cancer Identification

## Datasets

All datasets are downloaded when the classification models runs, and are placed in the `/dataset`
directory at the root level of the cloned repository.

For more information, see: https://gdc.cancer.gov/about-data/publications/pancanatlas.

The datasets used for the project were:

- DNA Methylation (Merged 27K+450K Only)
- TCGA-Clinical Data Resource (CDR) Outcome
